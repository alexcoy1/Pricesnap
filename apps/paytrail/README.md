# PayTrail

**Commission software for salespeople and business owners.**

Create an account in your browser (same approach as PriceSnap) — no Supabase or cloud setup. Upload invoices, match your price list, calculate commission.

## Run locally

```bash
cd apps/paytrail
npm install
npm run dev
```

→ **http://localhost:3000** → **Create account**

## Deploy

Netlify site — base directory `apps/paytrail`. Optional: `ANTHROPIC_API_KEY` for invoice AI. See **[DEPLOY.md](./DEPLOY.md)**.
